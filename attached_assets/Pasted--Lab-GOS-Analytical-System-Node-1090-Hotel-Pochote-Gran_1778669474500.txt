### API Server (`artifacts/api-server`)

Express 5, port 8080. No request timeout (`server.setTimeout(0)`, `keepAliveTimeout = 0`).

**Routes:**
- `GET /api/healthz` — health check
- `POST /api/synthesize` — SSE streaming synthesis pipeline (L1 → L2 → L3)
  - Body: `{ question: string, voices: VoiceInput[] }`
  - Emits: `layer`, `group_start`, `group_done`, `group_error`, `critique_start`, `critique_done`, `master_token`, `done`, `fatal_error`
  - Keepalive: SSE comment `: ping` every 10s
- `POST /api/astrology/reading` — GOS natal chart overlay
- `GET /api/field` — live EM/seismic field Ψ(t) bridge (KAPPA Heartbeat Tracker)
- `GET /api/field/bulk` — multi-device KAPPA bulk health

## Networking / Proxy Architecture

**Development routing** (Replit):
- Replit routes `/` → kappa-lab Vite (port 25135)
- Replit routes `/api` → api-server (port 8080) **directly**
- Frontend calls `/api/synthesize` directly — bypasses Vite proxy entirely
- This avoids Replit's 60-second reverse-proxy hard timeout on Vite connections
- The old `/proxy-api` Vite proxy still exists for other routes but is NOT used for synthesis

**Keepalive strategy:**
- API server sends SSE comment `: ping` every 10s
- Node HTTP server has no timeout (`server.setTimeout(0)`)
- Synthesis typically completes in 90–180s
