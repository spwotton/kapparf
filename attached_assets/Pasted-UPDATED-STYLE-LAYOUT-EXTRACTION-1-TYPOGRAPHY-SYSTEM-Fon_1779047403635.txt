UPDATED STYLE & LAYOUT EXTRACTION**

**1. TYPOGRAPHY SYSTEM**

*Font Families*
- **Serif**: EB Garamond (400), Lora (400) - Editorial/content use
- **Sans-Serif**: Montserrat (400, 700, 900), Open Sans (400, 600, 700), Roboto (400, 500, 700) - UI/Interface use
- **System Fallback**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell

*Font Features*
- Variable font support: `font-stretch: 100%` (Open Sans, Roboto)
- Font display strategy: `swap` (FOUT prevention)
- Weights: Light (300), Regular (400), Medium (500), Semi-bold (600), Bold (700), Black (900)

**2. DESIGN TOKENS**

*Color Palette*
- **Primary Accent**: `#059669` (emerald green), `#10b981` (trend up), `#ef4444` (trend down/red errors)
- **Calendar Blue**: `#4f99ff` (selection), `#f64747` (today marker)
- **Neutrals**: `#1e293b` (text primary), `#64748b` (text secondary), `#94a3b8` (text muted), `#f8fafc` (bg secondary), `#e2e8f0` (borders)
- **Dark Mode**: `#0f172a` (bg primary), `#1e293b` (bg secondary), `#334155` (borders)

*Spacing & Sizing*
- **Border Radius**: 8px (widgets/cards), 5px (calendar), 50px (pill buttons/calendar selection), 2px (small UI elements)
- **Shadows**: `0 4px 6px -1px rgba(0,0,0,0.1)` (widget elevation), `0 3px 13px rgba(0,0,0,0.08)` (calendar)
- **Gaps**: 2px (chart bars), 4px (tight spacing), 8px (standard gaps)

**3. COMPONENT ARCHITECTURE**

*Flatpickr Calendar*
- **Layout**: Flexbox-based 7-column grid (14.2857143% width per day)
- **Dimensions**: Fixed width 307.875px, max-height 640px
- **States**: 
  - Selected range: Blue background (`#4f99ff`), white text, 50px border-radius (pill shape)
  - Today: Red border (`#f64747`), red background on hover
  - In-range: Gray background (`#e9e9e9`) with connecting shadows
  - Disabled: 30% opacity, not-allowed cursor
- **Navigation**: SVG arrow icons, month dropdown (menulist appearance), year input (textfield)
- **Time Picker**: 40px height, flex layout with separators, AM/PM toggle

*Currency/Widget Components*
- **Container**: White/light background, 8px border-radius, shadow elevation
- **Input Fields**: Gray background (`var(--bg-secondary)`), 1px border, 8px radius, bold text
- **Buttons**: 
  - Currency selector: Flex layout with gap, dropdown arrow indicator
  - Swap button: Circular (50% radius), emerald background, white icon
- **Dropdown**: Fixed positioning (fullscreen overlay), scrollable list, search input at top
- **Chart**: Flex-aligned bars at bottom, emerald color (`var(--accent)`), 85% opacity, 4px min-width

**4. LAYOUT BEHAVIORS**

*Responsive Patterns*
- **Mobile Breakpoint**: 640px max-width (flex-wrap for subscribe form)
- **RTL Support**: 
  - Direction reversal for currency options (`flex-direction: row-reverse`)
  - Text alignment right for inputs
  - Dropdown arrow margin adjustments
  - Fixed positioning respects RTL contexts

*Theme System*
- **CSS Variables**: 
  - `--bg-primary`, `--bg-secondary`, `--border-color`
  - `--text-primary`, `--text-secondary`, `--text-muted`
  - `--accent` (themable brand color)
- **Dark Class**: `.dark` modifier switches color values
- **High Contrast**: Focus states with outline removal (relying on border/color changes)

**5. INTERACTION STATES**

*Hover Effects*
- Calendar days: Gray background (`#e9e9e9`), pointer cursor
- Navigation arrows: Red color (`#f64747`) on hover
- Input wrappers: 5% black background overlay (`rgba(0,0,0,0.05)`)
- Currency options: Secondary background highlight

*Focus States*
- Inputs: Outline removed (replaced with border-color changes)
- Calendar: Focus ring removed (`outline: 0`), relying on visual selection states

*Animations*
- Calendar open: `fpFadeInDown` 300ms cubic-bezier(0.23, 1, 0.32, 1)
- Transitions: Fill colors 0.1s (SVG paths), background 0.2s

**6. UTILITY PATTERNS**

*Reset Styles*
- `* { box-sizing: border-box }` universal border-box
- Number inputs: Spin buttons hidden (`appearance: textfield`)
- Date inputs: Calendar picker indicator made transparent/absolute positioned
- Buttons: Borderless, background-transparent, cursor-pointer

*Accessibility*
- `user-select: none` on navigation elements
- `touch-action: manipulation` on calendar (prevents zoom delays)
- `readonly` inputs with pointer cursor for date pickers
- `disabled` states with reduced opacity and `not-allowed` cursor

**7. GRID SYSTEM (Calendar Specific)**
- **7-Column Flex Grid**: Days distributed via `flex-basis: 14.2857143%`
- **Week Layout**: `.flatpickr-weekdaycontainer` flex container
- **Month Grid**: `.dayContainer` with `flex-wrap: wrap`, `justify-content: space-around`
- **Edge Handling**: First/last column border removal, row border collapsing
