### 3.16 Error Response Format

```json

}
```

Validation errors (from Zod):
```json
{
  "error": [{ "path": ["fieldName"], "message": "Error description" }]
}
```

HTTP Status Codes:
- `200` — Success
- `201` — Created
- `204` — No Content (deletes)
- `400` — Bad Request (validation, missing fields)
- `404` — Not Found
- `409` — Conflict (pipeline already running)
- `500` — Internal Server Error
