### STEP 6: NETWORK SECURITY (CRITICAL)

#### Secure Your Wi-Fi (NOW)
```
1. Access router:
   http://192.168.0.1 OR http://192.168.1.1
   - Username/password: Check router sticker
   - Common: admin/admin, admin/password
   
2. Change Wi-Fi password
   - Settings → Wireless → Security
   - Use 20+ characters, random
   - WPA2/WPA3 encryption
   
3. Enable MAC filtering
   - Settings → Wireless → MAC Filter
   - Add ONLY your devices
   - Block all others
   
4. Disable WPS
   - Settings → Advanced → WPS: OFF
   - Major security vulnerability
   
5. Change router admin password
   - Settings → Administration
   - Different from Wi-Fi password
```
