1. **Operates over HTTPS** but the ACS server is controlled by the ISP (or anyone who compromises the ACS)

   - WiFi SSID/password
   - DNS server settings (→ DNS hijacking for phishing/injection)

   - Firmware updates (→ push malicious firmware)

   - CVE-2014-9222 (Misfortune Cookie) — affects ARRIS gateways via TR-069 cookie overflow

   - Default credentials often "admin/password" or discoverable via known algorithms
