            'ble': deque(maxlen=100)

    def on_wifi_packet(self, timestamp):

        if len(self.windows['wifi']) == 1024:

## 8. Operational Security (Zero-Signature)

Since you're using **passive sniffing only** (LTESniffer, Sniffle, KiwiSDR remote), you emit **zero RF energy**. However:

- **Tor routing:** All MQTT traffic → Tor hidden service (your analysis server is .onion)
- **MAC randomization:** Your laptop WiFi uses random MACs (Windows/Linux feature) to avoid being fingerprinted by the very system you're monitoring
- **Decoy traffic:** Use `WindowsSpyBlocker` hosts file to block telemetry, preventing your own device from being a data point in your own database
