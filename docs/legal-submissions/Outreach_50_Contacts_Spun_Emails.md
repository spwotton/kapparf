## SEND DISCIPLINE

**Rule 1 — Never identical:** Within the same category, vary at least the opening sentence and the closing question when sending to multiple contacts.

**Rule 2 — One per session:** Do not send all 50 in one sitting. Batch in groups across different days and networks.

**Rule 3 — Network hygiene:** Send from mobile data (not home WiFi) given documented MITM. Use webmail in a fresh browser session, type fresh — do not paste directly from this file on a compromised device.

**Rule 4 — Log what was sent:** Note date, contact, and which template variant was used. When a reply comes in, the sender and subject make it traceable.

**Rule 5 — Priority order (recommended):**
- T+0: CSIRT-CR (17), OIJ Tech Crimes (28), OIJ CICO confidential (29) — fastest to act on network intrusion
- T+1: SUTEL Spectrum (23), MICITT Radio (16) — spectrum enforcement
- T+2: DGAC Director General (5), AERIS Operations (2) — aviation safety
- T+3: ICE Security (36), ICE Power (40) — infrastructure owner
- T+7: DIS (25), Contraloría (46, 47), Procuraduría (48) — systemic/legal
- T+14: Presidencia (49, 50), COCESNA (11–14) — escalation only if no replies
